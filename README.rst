=========
 Hacking
=========

Source code
===========

Get the source code::

   $ hg clone https://github.com/icemac/icemac.ab.calendar

or fork me at: https://github.com/icemac/icemac.ab.calendar

Running the tests
=================

To run the tests yourself call::

  $ virtualenv-2.7 .
  $ bin/pip install zc.buildout
  $ bin/buildout -n
  $ bin/py.test
